# lvrpiz Assets

Este repositorio contiene los materiales, presentaciones y recursos que utilizo en mi blog y charlas.  
Aquí encontrarás **PDFs, demos y documentos** que comparto públicamente para que puedas revisarlos, estudiarlos o reutilizarlos.

---

## Contenido
Cada carpeta corresponde a una charla, intervención o sección de contenido.  
Los archivos están disponibles para descarga directa desde GitHub o mediante enlaces `raw`.

### 1️⃣ IA para desarrolladores
- **Archivo principal**: `1_IA_para_desarrolladores.pdf`
- **Subcarpetas**:
  - `ide`: Recursos sobre entornos de desarrollo.
  - `mcp`: Material sobre mejores prácticas de código.
  - `mlops`: Introducción a MLOps.
  - `modelos-locales`: Uso de modelos de IA en local.

### 2️⃣ IA irresponsable
- **Archivos principales**:
  - `2_IA_irresponsable.pdf`: Presentación principal.
  - `2_IA_irresponsable_notas.pdf`: Presentación con notas adicionales y enlaces.
- **Otros recursos**:
  - `LLMIntro.excalidraw`: Diagrama introductorio a modelos de lenguaje.
  - `synthid_text.ipynb`: Notebook sobre generación de texto.
  - `watermark_image_embed.ipynb`: Notebook para incrustar marcas de agua en imágenes.
  - `watermark_image_detect.ipynb`: Notebook para detectar marcas de agua.
