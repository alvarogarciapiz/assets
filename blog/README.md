# Assets · Blog / Newsletter

Este directorio forma parte del repositorio **assets**.

Aquí encontrarás **todas las entradas de mi blog/newsletter** disponibles en **Markdown y PDF**, para que puedas **descargarlas y estudiarlas**.

## Qué hay aquí

- Entradas organizadas **por orden cronológico**
- Cada entrada está separada en **carpetas**
  - `markdown/` → versión editable
  - `pdf/` → versión lista para leer o archivar

## Sobre la newsletter

**“Análisis técnicos para profesionales que no tienen tiempo”**  
Por **Álvaro García Pizarro**

No es otra newsletter de noticias genéricas.  
Cada semana recibes un **Blueprint**:  
un análisis técnico profundo sobre **Tecnología e Ingeniería**, diseñado para:

- Obtener conocimiento **práctico y aplicable**
- Aprender tecnologías nuevas con **ejemplos reales**
- Ir al grano, sin relleno
